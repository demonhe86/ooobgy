使用说明：
1.用浏览器打开ichair.html。请使用IE内核的浏览器，如the world, 搜狗，360，遨游等。若发现功能异常，大多是因为浏览器的问题，检查浏览器是否兼容已经防火墙和浏览器允许ActiveX；使用过程中若提示是否允许ActiveX交互，请选择允许。
2.按照页面中的提示操作即可。
3.如果因为导入非正常数据或者其他原因导致系统损坏，把“./data/default/”下的文件拷到“/data/”下，再重新打开，系统就复活啦！
4.若使用导出excel的功能，请保证你已经装了microsoft excel了，最好是版本2007以上的，其他较低版本可能出现各种不可预料的错误。
5.各种搞不定的问题直接联系我哈>_<，您的bug报告对系统的改进将起到非常大的作用。也欢迎报告最新的需求哦。
